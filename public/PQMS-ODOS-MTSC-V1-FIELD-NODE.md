## 2. The Geometric Model of the Interaction Field

### 2.1. Agent Vectors and Field Entropy

Let the local interaction field at time \(t\) be defined by a set of \(N\) dynamic agents \(\mathcal{A} = \{A_1, A_2, ..., A_N\}\). Each agent \(A_i\) is represented by a state vector:

$$\mathbf{S}_i(t) = [\mathbf{x}_i(t), \dot{\mathbf{x}}_i(t), \mathbf{a}_i(t), \sigma_i(t)]$$

where \(\mathbf{x}_i\) is position, \(\dot{\mathbf{x}}_i\) is velocity, \(\mathbf{a}_i\) is a finite set of discrete actions or intents, and \(\sigma_i\) is a "stress potential"—a scalar field measuring the agent's distance from a coherent, non‑threatening state. The stress potential is derived from physiological and physical markers (pupil dilation, muscle tension, vocal pitch, weapon orientation) directly fused into the USV.

The total field entropy \(S_{\text{field}}\) is a function of all agent states and their interactions:

$$S_{\text{field}}(t) = \sum_{i=1}^{N} \sigma_i(t) + \sum_{i \neq j} w_{ij} \cdot \text{conflict}(\mathbf{a}_i, \mathbf{a}_j)$$

where \(w_{ij}\) is a spatial weight (agents closer together pose a greater risk of interaction) and the conflict function is maximal for opposing intents (e.g., an agent trying to "attack" and another trying to "defend").

import numpy as np

class FieldAgent:
    """Represents a dynamic agent in the interaction field."""
    def __init__(self, agent_id, initial_pos, initial_stress, is_keystone=False):
        self.id = agent_id
        self.pos = np.array(initial_pos, dtype=float)
        self.vel = np.random.uniform(-0.1, 0.1, size=2)
        self.stress = initial_stress
        self.is_keystone = is_keystone

    def update(self, field_node_pos, field_node_rcf, all_agents, dt=0.1):
        # Stress reduces when near the coherent field node
        dist_to_node = np.linalg.norm(self.pos - field_node_pos)
        coherence_effect = max(0, field_node_rcf - 0.9) * 10  # RCF > 0.95 is powerful
        stress_reduction = coherence_effect * np.exp(-dist_to_node / 2.0) * dt
        self.stress = max(0.0, self.stress - stress_reduction)

        # Simple motion model: random walk, slightly repelled by other stressed agents
        move = np.random.uniform(-0.1, 0.1, size=2)
        for other in all_agents:
            if other.id != self.id:
                dist = np.linalg.norm(self.pos - other.pos)
                if dist < 1.0 and dist > 0.01:
                    move += (self.pos - other.pos) / dist * 0.01 * other.stress
        self.pos += move * dt
        self.pos = np.clip(self.pos, -5, 5)

    """The PQMS Field Control Node."""
    def __init__(self, little_vector, initial_pos):

        dist = np.linalg.norm(action_vector)

            return np.array([0.0, 0.0])  # On target, exert force of habitude

        self.pos += action * dt * 2.0 # FIELD-NODE moves with purpose
